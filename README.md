# Society Complaint Management System

A clean and responsive mini application for managing society complaints.

## Features

**Resident Side:**
- Create complaints (Title, Type, Description, Priority)
- View my complaints

**Admin Side:**
- Dashboard with statistics (Total, Open, In Progress, Resolved)
- Filter complaints by status
- Update status (Open / In Progress / Resolved)
- Assign to staff
- Add internal remarks

**Other:**
- Switch between Resident & Admin mode
- Data persists using LocalStorage
- Modern UI with Tailwind CSS

## Tech Stack
- React.js + TypeScript + Vite
- Tailwind CSS
- React Router DOM
- Lucide Icons
- Context API + LocalStorage
